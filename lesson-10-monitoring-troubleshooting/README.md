# Lesson 10: Monitoring and Troubleshooting

## Overview

This lesson covers monitoring, diagnostics, and troubleshooting in Microsoft Entra ID. Learn to use sign-in and audit logs, configure diagnostic settings, write KQL queries, create workbooks and alerts, and resolve common identity issues.

## Learning Objectives

By the end of this lesson, you will:
- Navigate and analyze sign-in and audit logs
- Configure diagnostic settings for log export
- Write KQL queries for security analysis
- Use built-in and custom workbooks
- Set up alerts and notifications
- Troubleshoot common authentication issues
- Monitor service health and Connect Health
- Automate log access with PowerShell and Graph API
