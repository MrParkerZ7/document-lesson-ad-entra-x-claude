DIAGRAM FORMAT REQUIREMENTS:
All diagrams must use curved lines with this style:
- Use `curved=1` instead of `edgeStyle=orthogonalEdgeStyle`
- Add `shadow=1` to the mxGraphModel
- Use `strokeWidth=3` for connector lines
- Use `endArrow=blockThin;endFill=1` for arrows
- Include bezier control points using `<Array as="points">`

Example curved arrow:
```xml
<mxCell id="arrow1" style="curved=1;endArrow=blockThin;endFill=1;strokeColor=#0078D4;strokeWidth=3;exitX=1;exitY=0.5;entryX=0;entryY=0.5;shadow=0;" edge="1" parent="1" source="source-id" target="target-id">
  <mxGeometry relative="1" as="geometry">
    <Array as="points">
      <mxPoint x="400" y="200" />
      <mxPoint x="450" y="250" />
    </Array>
  </mxGeometry>
</mxCell>
```
