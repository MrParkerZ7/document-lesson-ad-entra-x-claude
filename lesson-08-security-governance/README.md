# Lesson 08: Security and Governance

## Overview

This lesson covers security features and identity governance in Microsoft Entra ID. Learn how to protect identities with risk-based policies, implement just-in-time privileged access, conduct regular access reviews, and automate access lifecycle management.

## Learning Objectives

By the end of this lesson, you will:
- Configure Identity Protection for risk detection and response
- Implement Privileged Identity Management (PIM) for just-in-time access
- Set up Access Reviews for periodic access certification
- Use Entitlement Management for self-service access packages
- Monitor security with audit logs and Log Analytics
- Configure security alerts and automated responses
- Improve security posture using Identity Secure Score
- Apply governance best practices across all areas
